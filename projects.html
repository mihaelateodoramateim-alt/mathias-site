Selected work

Practical engineering, delivered with care.

Below is a summary of projects we’ve built or operated. Each one balances reliability, security and everyday usability.

Smart Home Automation

We designed and delivered end-to-end automation systems that join household devices into a cohesive whole. Lighting adapts to presence and daylight; heating and cooling react to real-time demand; scenes can be triggered by voice or schedule. The control layer is simple on purpose—one dashboard, clear states, quick overrides.

  • Voice integrations: Alexa & Google Home
  • Presence-aware lighting with manual fallback
  • Energy-aware HVAC control
  • Secure messaging between devices

Video Surveillance & Home Security

For private homes and small businesses we implemented HD IP surveillance with discreet placement and encrypted storage. Owners monitor their property from anywhere, receive meaningful alerts—not noise—and can link cameras with smart locks and alarms for full-chain protection.

  • Night-vision IP cameras and motion zones
  • Encrypted NVR with safe remote access
  • Mobile notifications tied to real events
  • Integration with access control

Home Datacenter

For advanced home users we engineered compact, quiet racks that provide enterprise-grade storage and compute at domestic scale. Centralized data, automated backups, VM hosting and media services—all secured, monitored and energy-efficient.

  • ZFS storage with snapshotting and off-site sync
  • VM and container platforms for apps and labs
  • Hybrid cloud integrations for burst and backup
  • UPS and thermal management for continuity

IoT Communication Platforms

We built lightweight messaging layers so devices can talk reliably without drowning the network. From garden irrigation to temperature and motion sensors, the system surfaces what matters and stays invisible the rest of the time.

  • MQTT-based messaging with ACLs
  • Edge processing for low latency
  • Dashboards with meaningful alerts
  • Security baked in from day one
